About

Halo, saya Ahmad ilham B. Seorang Mahasiswa yang tertarik dengan dunia pemrograman khususnya pemrograman web. Saya belajar pemrograman melalui mata kuliah yang saya dapatkan di kampus, lalu saya dalami secara otodidak dengan belajar di youtube ataupun mengikuti bootcamp. Sejauh ini saya sudah membuat beberapa aplikasi berbasis website sederhana, mulai dari prototype hingga aplikasi yang telah terhosting. Bergelut di dunia web programming kini telah lumayan memberikan penghasilan bagi saya.

Selain Coding, saya juga gemar mempelajari ilmu desain grafis. Sejak 2020 Silam, sejak awal Covid-19 melanda, saya menyibukkan diri saya di rumah dengan belajar Coreldraw sebagai tools saya untuk mendesain. Tidak hanya desain grafis, saya juga belajar video editing menggunakan adobe premiere pro. Hingga kini, dari 2 bidang ilmu diatas, saya juga lumayan mendapatkan penghasilkan dari hasil karya saya.

Tidak hanya fokus belajar secara akademik, saya juga gemar mengikuti kegiatan-kegiatan non akademik seperti berorganisasi. Selama berkuliah, saya telah mengikuti beberapa organisasi, diantaranya :

    Anggota Bidang Informasi , Publikasi dan Rumah tangga
  2. Himatika FMIPA Unhas (2020-2021)
    Koordinator Bidang Kerohanian Islam

  5. Relawan Pendidikan Indonesia (2021-2022)

  6. Permikomnas RI (2020 - Sekarang)
    Anggota

Terakhir saya akan sedikit bercerita tentang hobi yang biasa saya lakukan ketika merasa bosan dengan kehidupan saya wkwk. saya suka dengan kegiatan yang berhubungan dengan alam seperti pendakian, traveling, pantai dan perjalanan lainnya.
